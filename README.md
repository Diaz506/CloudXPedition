# ☁️ CloudXPedition

**Your expedition into the cloud starts here.** Complete challenges, earn XP, collect badges, and conquer 6 phases on your journey to becoming a cloud pre-sales pro. Start as a ☁️ Cloud Rookie, level up through 7 ranks, and prove your skills across IT fundamentals, infrastructure, cloud services, and pre-sales mastery. Track your progress, maintain your study streak, and finish as a ⭐ Cloud Legend.

---

## 🚀 Features

- **6 Learning Phases** — Structured path from IT basics to certification readiness
  - 🌐 Phase 1: IT Fundamentals
  - 🛡️ Phase 2: Infrastructure & Identity
  - ☁️ Phase 3: Cloud Fundamentals
  - ⚙️ Phase 4: Cloud Services Deep Dive
  - 💼 Phase 5: Pre-Sales Skills
  - 📜 Phase 6: Certification & Portfolio
- **Gamification Engine** — XP points, 7 level ranks, and 11 collectible badges
- **Self-Assessment** — Evaluate your current cloud knowledge before starting
- **Progress Dashboard** — Track completion, badges earned, and study streaks
- **Curated Resources** — Handpicked learning materials with category filtering
- **Local Progress** — All progress saved in your browser (localStorage)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Diaz506/CloudXPedition.git
   ```
2. Open `index.html` in your browser — no build step or server required.
3. Take the **Self-Assessment** to find your starting point.
4. Begin your expedition through the 6 phases!
